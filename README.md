## **ğŸ’» Cara Install & Menjalankan (Localhost)**

Ikuti langkah-langkah ini untuk menjalankan proyek di komputer kamu:

### **1\. Persiapan Lingkungan**

Pastikan kamu sudah menginstall **XAMPP** (atau aplikasi server lokal sejenis) dan **Git**.

### **2\. Clone Repository**

Buka terminal (Git Bash atau CMD) di folder htdocs kamu, lalu jalankan:

cd D:\\xampp\\htdocs  
git clone \[https://github.com/kelfinjonio12-sudo/LaskarTrip.git\](https://github.com/kelfinjonio12-sudo/LaskarTrip.git)

### **3\. Konfigurasi Database**

1. Buka **XAMPP Control Panel** dan nyalakan module **Apache** dan **MySQL**.  
2. Buka browser dan akses http://localhost/phpmyadmin.  
3. Buat database baru dengan nama laskartrip\_db (atau sesuaikan dengan nama di file koneksi kamu).  
4. Import file database (biasanya berformat .sql) yang ada di dalam folder proyek ini ke database yang baru dibuat.

### **4\. Jalankan Website**

Buka browser dan kunjungi alamat berikut:  
http://localhost/LaskarTrip
